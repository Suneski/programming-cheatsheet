JAVASCRIPT: VARIABLES

~high school math flashbacks!~

Variables are best thought of as containers that hold certain values. These values can then be referenced and/or used later in a program by referencing the variable.

Variables can hold various types of information inputs, include (but not limited to):

  • Strings
  • Numbers
  • Objects

To set up a variable in JavaScript, all you need to do is type "var" + the variable's name + "=" + the value being inputted + ";"

NOTE: When naming your variable, never start with a capitalized letter. Always start lower case. Additionally, a variable's name must NOT include any spaces or hyphens. Either use single-word variable names or camel case (ex: camelCase).

Example:

var superAwesomeVariable = 'My variable contains a string';

If you need to change your variable value, all you need to do is replace its value with a new value. You can also restate the variable, without "var" and with the new value.

You can combine variables inside of variables.

var pizzas = 5;

var slicesPerPizza = 8;

var totalSlices = pizzas * slicesPerPizza;

Once you set up these variable, if you run it in your console, you see your combined value.

You can even combine strings!
