WEB FONTS

There are several "Web Safe" fonts that most browsers can read (for a list of these fonts, click here).

But what if was want to use a different font? Maybe we want a font more elegant than "cursive," or maybe a font reminiscent of video games from the 80s. Other sites have been able to use far more varied fonts.

Some of my favorites include:

VT323

Press Start 2P

Orbitron

But how do you do such wizardry?! Well, there are a couple of methods.

(Just a quick note, when you use font-family with these fonts, include some "Web Safe" fonts as back up, in case there is a server issue.)

@font-face

When Best to Use This Method: When using a font from your own server/computer

How to Use: In your CSS file, type "@font-face" followed by your curly brackets. Inside of your brackets, you will need to include "font-family:" with the name of the font, as well as "src: url()" with the font file's location in your parenthesis.

From there, just use "font-family:" and the font-name anywhere in your CSS file.

NOTE: @import must be included at the top of your CSS file.

@import

When Best to Use This Method: When using a font from an external server

How to Use: In your CSS file, type "@import url('')" with the URL for the font between the single-quotes. You do not use curly brackets when using @import.

Once your @import has been set up, you can just use "font-family:" and the font-name anywhere in your CSS file.

Many sites that offer fonts, including Google Fonts, will have the @import line ready for you to copy/paste.

NOTE: As with using @font-face, @import must be included at the top of your CSS file.

<link>ing in HTML"

When Best to Use This Method: When you'd rather add the font via HTML than CSS

How to Use: In your <head> section, create a <link> tag. Set an "href" to the font's URL, and set "rel" to stylesheet. From there, you can use "font-family:" with the font name in your CSS.

NOTE: Google Fonts, as well as other sites, will have the <link> line ready for you to copy/paste.
