Web Fonts

Sometimes, we want to use a font beyond those that are pre-installed on our computers. Maybe we want a font more elegant than "cursive," or maybe a font reminiscent of video games from the 80s. Other sites have been able to use far more varied fonts, but how?!

There are a couple of different ways to import fonts from the web, and I cover a few methods below.
