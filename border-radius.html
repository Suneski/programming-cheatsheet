BORDER RADIUS

With the use of the border-radius property, you can give elements rounded edges.

There are four property values that can be used in order to set the border-radius.

  • length: set a pixel amount (default is zero)
  • percent: percentage between 0 and 100
  • initial: set to default value
  • inherit: inherit from parent element

By default, border-radius will give all four corners the same curvature, but you can specify corners as necessary.

  1. One Value

    border-radius: 10px

  2. Two Values

    border-radius: 10px 20px

  3. Three Values

    border-radius: 10px 20px 30px

  4. Four Values

    border-radius: 10px 20px 30px 40px
