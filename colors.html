You can use different colors in various ways to bring your webpage to life. However, whether you're setting the font color with "color:" or setting a background color with "background-color:" you still need to select a color. But how???

Three Common Approaches

  • Name

    • Many colors can be summoned just by their name. Many of these colors tend to be common (RED, BLUE, GREEN), but others are rather obscure. (LIGHTSTEELBLUE, PALEGOLDENROD, PLUM).

    For a full list of colors that can be called by name, visit W3Schools' page on colors.

  • Hexadecimal Value

    • Colors can also be summoned by using their corresponding hexadecimal numbers.

    For example, the hexadecimal for the color red is #FF0000.

    With hexadecimal notation, you can get colors that are more specific than if you just use a color's name. You can experiment with various shades.

    If you use #ff7777, you get a lighter shade of red.

  • RGB Value

    • Another way to classentify a color is with its RGB value (RGB stands for Red-Green-Blue).

    By combining varying intensities of red, green, and blue, new colors can be attained. The "intensity scale" ranges from 0 to 255.

    The RGB value for (standard) blue is rgb(0, 0, 255). The values for red and green are set to zero. But if you sprinkle some red and green, you get a new color (example: rgb(145, 54, 255)).
